Sobre o Git

Git é um sistema de controle de versão.

Na prática, ele permite:

  • voltar no tempo
  • trabalhar em equipe
  • testar sem quebrar o projeto

Por que o Git existe

Antes do Git, desenvolvedores tinham vários problemas:

  • arquivos duplicados (versão-final, versão-final-2…)
  • conflitos ao trabalhar em equipe
  • dificuldade para desfazer erros

História do Git

Em 2005, Linus Torvalds (criador do Linux) criou o Git para resolver esses problemas.

O motivo da criação

Na época, o desenvolvimento do Linux usava uma ferramenta chamada BitKeeper.

Só que aconteceu um problema:

  • a licença foi retirada
  • comunidade ficou sem ferramenta

A solução

Linus Torvalds criou o Git em poucos dias com alguns objetivos:

  • ser extremamente rápido
  • funcionar de forma distribuída
  • ser seguro
  • permitir trabalho em equipe

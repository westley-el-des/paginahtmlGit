Oque é git

Git é um sistema de controle de versão.

Em vez de ficar salvando:

  • projeto1
  • projeto2
  • projeto3

O Git salva tudo automaticamente e permite:

  • voltar no tempo
  • trabalhar em equipe
  • testar sem quebrar o projeto

Conceito MAIS importante (entenda isso primeiro)

Git funciona com 3 etapas:

  • Working Directory → onde você edita os arquivos
  • Stage (Index) → onde você prepara o que vai salvar
  • Repository → onde fica salvo de verdade

3 comandos principais do Git

git init

Serve para iniciar o Git em um projeto

  • É o primeiro passo
  • Cria um “repositório” no seu projeto
  • A partir daqui o Git começa a acompanhar tudo

git add

Serve para preparar o que vai ser salvo

  • É o segundo passo
  • Adiciona arquivos para a “Stage” (Index)
  • É como se fosse um carrinho de compras

git commit

Serve para salvar de verdade

  • É o terceiro passo
  • Salva o que está na “Stage” para o “Repository”
  • É como se fosse o caixa do supermercado
